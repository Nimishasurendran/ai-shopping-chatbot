The AI Shopping Chatbot is a web-based application that allows users to interact with a chatbot to browse products, add items to a cart, and place orders through a conversational interface. The chatbot understands user messages and responds intelligently, making the shopping experience simple and user-friendly.
This project is built using Python and FastAPI for the backend and HTML, CSS, and JavaScript for the frontend. 

FEATURES:
View available products with prices
Add products to cart using chat messages
View cart details and total price
Place and confirm orders
Simple and attractive web interface
FastAPI-based backend with REST APIs

Technologies Used
Python
FastAPI
HTML, CSS, JavaScript
Jinja2 Templates
JSON for product data

The objective of this project is to demonstrate how AI-driven chat interfaces can be used in e-commerce applications to improve user interaction and automate basic shopping tasks.
